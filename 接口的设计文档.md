## 配置考试类型

### 接口定义

/api/conf/listExamType

### 返回数据

```json
[{
  name: "全部",
  wsType: 0
},{
  name: "四六级",
  wsType: 2
},{
  name: "考研",
  wsType: 1
},{
  name: "雅思",
  wsType: 3
},{
  name: "托福",
  wsType: 4
}]
```

**注意事项**

* 除非服务器内部错误，否则肯定会有数据返回
* 数据已排序
* 类型“全部”的wsType不使用，如果选择全部，则不传wsType参数

### 详细设计

```
配置数据存入redis，同时代码里写死一份数据，数据直接存储json字符串
if redis里没读出数据
    使用代码里写死的数据
else redis里有数据
    if 代码里写死的和redis里不同
        更新本地数据，与redis相同
    else
        直接返回数据
```
